Вопрос 5

Алгоритм Кнута-Морриса-Пратта

Представим есть строка "аабаа" и мы её ищем в строке "аабааббаббаабаабб". Теперь нам нужно склеит строки поместив между ними символ, который НЕ ВСТРЕЧАЕТСЯ ни в одной из строк. Это важно, чтобы символ разделения не смешался ни с одной из строк и максимальные значения префикс-функции были равны длине эталонной строки. Например, "аабаа#аабааббаббаабаабб". Теперь передадим полученную строку в префикс-функцию. Получим массив 

[0, 1, 0, 1, 2, 0, 1, 2, 3, 4, 5, 3, 0, 0, 0, 0, 1, 2, 3, 4, 5, 3, 0].

В массиве получены два значения равных длинне эталонной строки. Это означает, что всего в строке 2 вхождения эталонной строки, а их позиции можно подсчитать как индекс значения их длин минус их (длина - 1).

Алгоритм Кнута-Морриса-Пратта (KMP) является алгоритмом поиска подстроки в строке. Он был разработан Дональдом Кнутом, Джеймсом Моррисом и Вацлавом Праттом в 1977 году и широко применяется в обработке строк, алгоритмах компиляции и других областях.
